# 🚀 Quick Start: Create Admin User

## Easiest Way (3 Steps):

### 1. Run your app
```bash
flutter run
```

### 2. Click the "Create Admin" icon
- Look for the green person icon (➕👤) in the top-right corner of the landing page
- Click it to open the Admin Setup page

### 3. Click "Create Admin Account"
- The button will create your admin user automatically

## Where to Login:

1. Click the Admin Panel icon (⚙️) in the top-right corner
2. Login with the credentials above
3. You're now logged in as admin!

---

## Alternative: Run Database Seeder

If you want to create admin + sample data (users, recipes, settings):

1. Navigate to the Database Seeder page (you can add this route to your main.dart)
2. Click "Run Seeder"
3. Wait for completion
4. Login with the same credentials above
